1:reentrantLock
2:reentrantReadWriteLock

some important interfaces:
1: Lock
2: ReadWriteLock
3: Condition

two (abstract) classes:
1: AbstractOwnableSynchronizer
2: AbstractQueueSynchronizer

for the util lock,
1: CountDownLatch
2: CyclicBarrier
3: Semaphore
